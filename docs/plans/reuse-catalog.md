# Reuse catalog — blog-studio-lambda → app-starter/nexus-ui

Source: `github.com/achammah/blog-studio-lambda` (a fork of this starter). Goal: generalize every reusable unit back here, config-driven, so every future app inherits it. Each lane owns a category below; overlaps with already-shipped features are RECONCILED (take the better, never double-add). Base tag: `base-reuse-a`.

## Lanes
- **foundations** — Nexus plumbing + async + tokens (dependency root): `RemoteStore.sync()` (external-writer catch-up, verbatim), warehouse `_read`/`loadSince`, `emulatorChat()` (generalize `/api/copilot` session proxy) in `src/lib/nexusClient.mjs`, `runAiTask()` (generalize `redlineArticle`) in `server/aiTaskRunner.mjs`, `fireAsyncGeneration()` (generalize `/api/generate-article`) in `server/asyncGeneration.mjs`, `api.ts` per-call `timeoutMs`, env conventions (`*_TASK_ID`/`*_DEPLOYMENT_ID`/`*_WEBHOOK_URL`, NO live defaults). Hooks → `src/ui/hooks/`: `usePollRev` (move), `useAsyncOp(predicate,{pollFn,everyMs,stallAfterMs,onSettle})`, `useDebouncedSave(persistFn,delay)→{saveState,trigger}`, `ThinkingDots` primitive. Motion → `src/ui/tokens/`: `--nx-ease-settle`/`--nx-ease-spring` tokens, `motion.css` (riseIn/popIn families + ONE `@media(reduce)` guard via `[data-motion]`, `.nx-tap-scale`/`.nx-hover-lift`). Opt-chip palette as a skin-overridable preset.
- **editor** — `NotionEditor` → `nexus-ui/src/record-core/`; new `richText` FieldType (value=`Block[]`); `RecordPage` render branch (parallel to longText); DataTable cell preview; `textToBlocks`/`blocksToMarkdown`/slash-menu/`BlockHandle`/image-drop. Entity-agnostic — lift clean + wire the field type.
- **suggestions** — `useSuggestions(blocks,onChange,changes,onChange)` (extract Redline accept/reject/undo/all engine); inline redline widget + right-rail panel; field-level `suggestTaskId` + templated `POST /api/objects/:obj/:id/suggest/:field` (uses foundations `runAiTask`); generic record **state-pipeline** component (`Pipeline`+`Chip`). Needs editor + foundations.
- **copilot** — `Copilot` panel + `CopilotButton` + toggle (⌘I/`c`) + tool-chips; markdown renderer (`inlineMd`/`renderMd`/`renderAgent`) → shared `renderMarkdown` primitive; `copilotContext` seam (config `contextFields[]` per object, replaces hardcoded `articles`/hash-sniffing); `copilot:{deploymentId,title,mark,emptyStateCopy,suggestions}` config; SUPERSEDE ChatDock (iframe→fallback); REWRITE dead `chatBridge.ts` to the proven `emulatorChat` shape. Needs foundations `emulatorChat`.
- **filters** — `Filters.tsx` (bar+chips+`matchFilters`+`opsFor`) → `nexus-ui/record-core/`; `filterableFields()` helper; ObjectView wiring + persistence; typed `ObjectConfig.columns?:string[]` default-visibility. RECONCILE grid-keynav (our W39) + ImportDialog (our import-apikeys) — take better, don't double-add.
- **wizard** — `Wizard`/`Stepper` engine + `NewArticleModal`→`ModalOverlay` + `ChipListInput` + `SourcesInput`/`FileList` → `nexus-ui/src/blocks/wizard/`; guided-vs-blank landing. App supplies questions + onComplete/onCreateBlank.
- **mobile** — bottom tab bar (`.mobileNav`, off config.objects, coexists with existing drawer), config-driven `goChords` map, `ShortcutsOverlay`+split core/app shortcuts, keyboard-nav layer, `MobileReviewBanner` primitive (generic step-through N items), `side--icons`/`--peek` collapse, iOS-zoom/tap conventions. RECONCILE with nav-mobile wave.
- **settings** — `SettingsPage` tab shell (`TABS:{key,label,icon,render}[]`) + `EditableRuleList` primitive (objectKey + field-shape + severity-enum as props) → nexus-ui; profile renderers stay app-side as example.
